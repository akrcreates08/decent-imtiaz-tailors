# Lux Studio Photography Website

## Overview
Lux Studio is a premium photography business offering a range of services tailored to capture life's special moments. This project is a modern, one-page website designed to showcase the studio's offerings, portfolio, and contact information.

## Features
- **Hero Section**: Bold headline with a WhatsApp call-to-action.
- **Services Section**: Display of services offered with sample pricing.
- **Portfolio Gallery**: Showcase of sample images from various photography sessions.
- **About Section**: Information about the studio along with a photo.
- **Instagram Preview**: Display of recent Instagram posts.
- **Contact Section**: Easy access to contact options including WhatsApp, Call, and Instagram.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd lux-studio-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `index.html` in your web browser to view the website.

## Customization
- **Text and Images**: Edit the content in the `src/partials` directory to update text and images.
- **Pricing**: Update service pricing in `src/data/services.json`.
- **SEO**: Modify SEO settings in `src/data/seo.json`.

## Deployment
To deploy the website, run the following command:
```
bash tools/deploy.sh
```
